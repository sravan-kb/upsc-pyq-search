# UPSC PYQ Search

**Connect with me:** [LinkedIn](https://www.linkedin.com/in/sravan-branwal) • [GitHub](https://github.com/sravan-kb)

A fast and lightweight search engine for UPSC Previous Year Questions, built with FastAPI and PostgreSQL.

🔗 **Live:** [upsc-pyq-search.onrender.com](https://upsc-pyq-search.onrender.com)

---

## Features

- Keyword-based search across PYQs
- Highlights matching keywords in results
- Relevance score ranking
- Mobile-friendly UI
- Fast and lightweight — no JS frameworks

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Deployed on Render

## Current Dataset

- UPSC Prelims 2025 (Paper 1)

> ⚠️ UPSC question papers are unstructured data — OCR processing sometimes causes questions to appear jumbled or split incorrectly. The search is fully functional but some question text may look irregular. This will be fixed over time.

## Known Issues

- Questions with tables (e.g. match the following) lose their formatting after OCR — they appear as plain text which is hard to read. This needs to be fixed.
