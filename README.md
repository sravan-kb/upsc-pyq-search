# UPSC PYQ Search Engine

A searchable UPSC Previous Year Questions search engine built using FastAPI and PostgreSQL.

## Features

- Search UPSC PYQs using keywords
- Highlight matching keywords
- Match score ranking
- PostgreSQL-based search
- OCR processed question papers
- Fast and lightweight UI

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Python
- OCR Pipeline

## Current Dataset

- UPSC Prelims 2025

## Future Improvements

- Add 2024 and older PYQs
- Subject filters
- Year filters
- Better ranking
- Full-text search
